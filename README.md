# AI Agents Configuration

Personal configuration for AI agents

## Installation

1. **Clone:**

```bash
git clone https://github.com/OscSer/agents-config.git
cd agents-config
```

2. **Configure:**

```bash
echo 'OPENROUTER_API_KEY=your-openrouter-api-key' > .env
```

3. **Install:**

```bash
python3 install.py
```

## Structure

```
├── claude/          # Claude Code configuration

```
